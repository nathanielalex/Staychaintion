# Chatbot Module Documentation

## Overview
The Chatbot module provides an Internet Computer actor that handles chat interactions, stores conversation history, and returns appropriate responses to user queries.

## Actor: Chatbot

### State Variables

#### chatHistory
- Type: `List.List<(Text, Text)>`
- Description: A stable variable that stores the history of conversations as a list of message-response pairs.
- Persistence: Stable across upgrades

### Public Functions

#### chat
- Description: Processes user input messages and generates appropriate responses
- Params: `message: Text` - The user's input message
- Returns: `async Text` - The chatbot's response to the user's message
- Behavior: 
    - Logs the user's message to the console
    - Generates a standard response
    - Adds the message-response pair to the chat history
    - Returns the response to the caller

#### getHistory
- Description: Retrieves the complete history of chat interactions
- Params: None
- Returns: `async [(Text, Text)]` - An array of message-response pairs representing the chat history
- Query: This is a query function that does not modify state
- Behavior:
    - Converts the internal List-based chat history to an array
    - Returns all stored conversations

## Dependencies
- Debug: Used for console logging of messages
- List: Used for managing the chat history data structure
